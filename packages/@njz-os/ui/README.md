# @njz-os/ui

Design tokens and (eventually) component primitives for NJZ RAT-OS.

## Surface (Phase 0)

- `tokens/colors` — semantic palette + per-module accents
- `tokens/space` — 4 px grid scale + radius
- `tokens/typography` — Space Grotesk display, Inter body, type scale

## Roadmap

- Phase 1: Button, Panel, Toggle, Checkbox, Card, Modal, Drawer, Toast primitives.
- Phase 2: Storybook publication, visual regression baseline.

## Extending Upstream

Once `@njz/ui` (upstream in ZeSporteXte) stabilises, re-export selected primitives here and document any RAT-OS overrides.
