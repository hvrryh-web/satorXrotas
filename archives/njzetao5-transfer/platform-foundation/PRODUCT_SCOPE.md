# Product Scope

## Vision
Deliver a PixelOffice-inspired web platform that makes AI-agent project management feel social, playful, and intuitive for new developers.

## In-Scope Products
1. **WebApp**
   - Planner board
   - Project timeline and sprint views
   - Agent assignment and status management
2. **Website Platform**
   - Public pages (landing, docs, pricing, release notes)
   - Auth gateway and user onboarding funnel
3. **Browser Extension**
   - Lightweight assistant panel for project context capture
   - Action forwarding into planner and task queue
4. **AI Interaction Layer**
   - Prompt orchestration
   - Tool-call history and agent action audit trail
5. **Pixel Sprite Agent UX**
   - Character states (idle, advising, warning, celebrating)
   - Conversational helper overlays tied to project state

## Core User Outcomes
- New developers can start contributing in under one day.
- Users can plan, assign, run, and review AI-assisted work in one workflow.
- Users understand agent behavior through visual and narrative feedback.
