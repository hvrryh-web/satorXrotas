# NJZetao5

NJZetao5 is the core webapp repository for a multi-agent collaboration ecosystem.
It is intended to bridge agent orchestration, team operations, and long-context knowledge workflows inside one centralized networked platform.

**Suggested GitHub repository description:**
> Webapp platform for multi-agent orchestration, collaborative AI operations, and second-brain knowledge management across ACP/MCP-enabled workflows.

## Mission
- Provide a central coordination layer for multi-agent planning and execution.
- Support conference-style multi-agent AI group chat as the minimum viable capability.
- Expand into a full AI ecosystem with persona libraries, collaborative RAG, and organization-ready operations.

## Current Repository Scope
- `platform-foundation/` - Product, architecture, standards, onboarding, and roadmap documents for the webapp platform.
- `godot-game-migration/` - Legacy and migration assets retained for reference.

## Product Capability Targets
1. **Agent collaboration and governance**
   - ACP/MCP-aligned agent workflows
   - Role/persona/scenario operating patterns
   - Shared logging and auditable action history
2. **Knowledge and context infrastructure**
   - Second-brain library with indexed context assets
   - File/data pathfinding and vector-oriented retrieval design
   - Long-form context management for team decision support
3. **Orchestration and integrations**
   - LangChain-style coordination patterns
   - Integrations with Notion, Obsidian, Hermes, and related tools
   - Weekly context review routines across lead agents
4. **Delivery experience**
   - Game-quality-but-simple sprite UX direction for assistant characters
   - Enterprise-grade engineering quality, security, and operational readiness
