# Personas — NJZ RAT-OS

**Status:** Active
**Source:** PRD §2.2

Three user tiers. Engineering and design decisions should reference these explicitly.

---

## Primary — Balanced Achiever (Ages 18–35)

**Snapshot.** Students and early-career professionals seeking measurable self-improvement without app fatigue.

| Dimension | Profile |
|-----------|---------|
| Income | $25K–65K |
| Screen time | 3.5–5.5 hrs/day mobile |
| Tech comfort | High |
| Existing app stack | Already subscribes to 2–3 productivity/wellness apps |
| Pain | App fragmentation, subscription fatigue, "tools that don't talk to each other" |
| Aspiration | A single dashboard for productivity + wellness with clear ROI |
| Top modules | Focus Hero, Soundscapes, Distraction Blocker, Micro-Learning, Brain Training |
| Engagement | 20–45 min × 2/day (morning warm-up + evening focus session) |
| Streak sensitivity | High |
| Channels | Reddit (r/productivity, r/getdisciplined, r/study), TikTok #studytok, Discord study servers, college ambassadors, Product Hunt |
| Price sensitivity | Medium ($30–50/yr) |
| LTV estimate | $45–75 (18 months) |
| Network value | Medium (referrals) |

**Voice quote (composite).**
> "I'm tired of paying for Lumosity, Endel, Freedom *and* a Notion subscription. I want one app where my focus sessions actually do something, not just track minutes."

**Design implications.**
- Onboarding must show cross-module value within 2 minutes.
- Streak heat maps are a first-class UI element.
- Social sharing of progress (especially cognitive profile) drives word-of-mouth.

---

## Secondary — Creative Professional (Ages 25–45)

**Snapshot.** Remote workers, writers, designers, developers, freelancers. Already invested in productivity tools; receptive to a consolidation play.

| Dimension | Profile |
|-----------|---------|
| Income | $55K–120K |
| Devices | Laptop + phone + tablet |
| Existing stack | Notion, Spotify, Freedom, often Ulysses or Scrivener |
| Pain | Context switching, deep-work fragmentation, frustrated by subscription sprawl |
| Aspiration | Deep work as default; data-driven self-optimization; aesthetic appreciation |
| Top modules | Writing Space, Focus Hero, Distraction Blocker, Soundscapes, PolyCo.World (evening wind-down) |
| Engagement | 45–90 min × 1–2/day (midday deep work + evening relaxation) |
| Analytics use | High — checks dashboards weekly |
| Channels | Product Hunt, Hacker News, NaNoWriMo, writing subreddits, RemoteOK, Nomad List, Dribbble, Figma |
| Price sensitivity | Low ($50–80/yr); appreciates lifetime tiers |
| LTV estimate | $80–150 (24 months) |
| Network value | Medium (team adoption potential) |

**Voice quote (composite).**
> "I want a writing tool that doesn't feel like a productivity tool that's also somehow a brain training app. Show me one place where my hours of deep work actually compound into something visible."

**Design implications.**
- Writing Space + Focus Hero must feel premium and uncluttered.
- Analytics depth is a meaningful upsell.
- PolyCo.World aesthetic resonates (RPG nostalgia from this cohort).

---

## Tertiary — Curious Learner (Ages 13–25)

**Snapshot.** Students and young gamers. Limited disposable income. Heavily social-driven. Free-to-play expectation is firm.

| Dimension | Profile |
|-----------|---------|
| Income | Limited disposable income |
| Devices | Mobile-first |
| Existing stack | Instagram, TikTok, Discord, Roblox, casual mobile games |
| Pain | Easily overwhelmed; school stress; needs short, guided interactions |
| Aspiration | Self-expression (avatar customization), social belonging, curiosity satisfaction |
| Top modules | PolyCo.World (exploration, decoration, social), Micro-Learning, Brain Training, Focus Hero (character progression) |
| Engagement | 15–30 min × 3–5/week |
| Channels | TikTok, Instagram Reels, YouTube Shorts, Discord gaming servers, school clubs, word-of-mouth from Primary tier |
| Price sensitivity | High (free-first) |
| LTV (direct) | $5–15 (6 months) |
| Network value | **High (viral growth)** |

**Voice quote (composite).**
> "I downloaded it because my friend's avatar looked cool in the screenshot. I stayed because the games are actually fun and my room got prettier the more I played."

**Design implications.**
- Free tier must offer real fun (not crippled).
- Screenshot/share affordances throughout PolyCo.World.
- Seasonal events drive return visits.
- Conversion to Primary tier monetization happens as they age (3–5 year horizon).

---

## Cross-Persona Insights

- All three tiers care about *visible progress*. The PolyCo.World metaverse is the rendering of that progress.
- All three are streak-sensitive, but Primary cohort is the *most* sensitive.
- The Tertiary tier is a growth engine; the Secondary tier is a revenue engine; the Primary tier is the heart.

## What This Doc Is Not

- Not a target-demographic exhaustive list. There are other valid users (parents, retirees, ESL learners) but Phase 1–2 focuses on these three.
