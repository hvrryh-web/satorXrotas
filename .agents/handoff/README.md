# .agents/handoff/

Session-to-session handoff notes. Drop a file when you close a session mid-task. Format `<topic>-YYYYMMDD.md`.

Each file is short (< 200 lines), focused on one topic, dated, and self-contained. Includes:

- What's done
- What's next
- Blockers / open questions
- Relevant file paths and line numbers

Remove the file once the next agent picks it up and resolves it (or moves it to `.agents/active/`).
