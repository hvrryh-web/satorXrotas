# .agents/active/

Drop a file here per actively in-flight branch or workstream. One file per topic, named `<branch-or-topic>.md`. Remove when merged/closed.

Each file contains:

- One-line description of what's being worked on
- Owner agent / role
- Channel (from COORDINATION_PROTOCOL.md)
- Link to PR(s) or session workplan(s)
- Blockers, if any
