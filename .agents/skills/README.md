# .agents/skills/

Per-agent and per-skill cheat sheets. Lives outside the universal AGENT_CONTRACT so quirks don't bloat the main contract.

Filename: `<agent-or-skill>.md`.

Examples:

- `codex.md` — tool-specific quirks for OpenAI Codex CLI
- `kimi.md` — quirks for Kimi K2
- `playwright.md` — how we run E2E in this repo
- `three.md` — Three.js conventions for `@njz-os/polyworld`
