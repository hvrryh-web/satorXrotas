# Support

| Question type | Best channel |
|---------------|--------------|
| How do I use feature X? | GitHub Discussions (once enabled) |
| I found a bug | GitHub Issues with `bug` label |
| I want to suggest a feature | GitHub Issues with `enhancement` label |
| I want to contribute | Read `CONTRIBUTING.md` |
| Security vulnerability | See `SECURITY.md` (do not file a public issue) |
| Press / partnerships | Maintainer email below |

## Maintainer

Repository owner: `@hvrryh-web`. Reach out via GitHub or the email associated with that account.

## Response Expectations

- Issues: triaged within one week.
- PRs from contributors: first review within one week.
- Security reports: acknowledged within 72 hours (see `SECURITY.md`).

We are a pre-launch project with a small team. Patience appreciated.
